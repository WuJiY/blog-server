# 博客后端

---

> 基于[koa2](https://github.com/koajs/koa)和[mongoose](https://github.com/Automattic/mongoose), 部署于腾讯云

## 安装

---

```shell
git clone https://github.com/zerowong/blog-server.git
cd blog-server
yarn install
yarn dev
yarn test
yarn start
```
